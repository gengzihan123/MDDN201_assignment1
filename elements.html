Branding Style

Logo, Fonts, and Color Pallete

Fonts

Primary Fonts

The primary font is Bodoni MT. The fonts are thick and thin, reminiscent of GPS precision, and as the title it is very clear.

secondary Fonts

The secondary font is arial. As a secondary font, it is mainly for the audience to be able to clearly identify better read. So I chose Arial as my secondary font

Logo

For the logo design, I have prepared three versions.

However, the first version is too simple, and the second version is too complicated, so I choose the third version as the logo.

Later, I considering that the webpage is mainly dark, so I changed the logo color to light blue.

Color palette

GPS is a high-tech system, for the color selection is given priority to with blue, with a dark blue background as a space. font colour use contrasting colours such as white for easy identification and emphasis.
